# Overview

This repository will hold many projects tailored to learn many essential concepts related to development such as
  POO, dependencies management & thid-party libraries, defining and using APIs, while learning Python language.

# General Rules

## Python projects
- Corrections will run on Ubuntu 20.04 LTS.
- Python version used for correction: Python 3.8.x.
- Every Python file must start exactly with:  
  `#!/usr/bin/env python3`
- Every Python file must:
  * Be executable.
  * End with a newline.
  * Be PEP8 compliant (pycodestyle 2.7.x).
  * Output must match expected formatting exactly.
  * No external libraries are allowed unless explicitly requested.
